Add testing related tasks to your Sakefile powered by [`sake-mocha`][sake-mocha]
and [`sake-chai`][sake-chai].

## Install
```bash
$ npm install sake-test --save-dev
```

## Usage
In your Sakefile:

```javascript
use('sake-test')
```
